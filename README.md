CNCsewingMachine

This project is intented to design a CNC embroidery machine with most of recycle parts. E.g., borken inject printers's rails and motors.

# hardware

* Embroidery hoop
sqaure shape hoop 
* X-Y axis moving platfrom
* arduino uno/raspberry Pi zeo
